# ☕ سیستم مدیریت کافه (Cafe Manager)

<div dir="rtl">

## 📋 معرفی پروژه

**سیستم مدیریت کافه** یک اپلیکیشن مدرن و جامع برای مدیریت کامل یک کافه یا رستوران است. این پروژه با هدف ساده‌سازی فرآیندهای روزانه مانند ثبت سفارش، مدیریت منو، پیگیری وضعیت میزها و گزارش‌گیری طراحی و پیاده‌سازی شده است.

## ✨ ویژگی‌ها و قابلیت‌ها

- 🍽️ **مدیریت منو**: اضافه، ویرایش و حذف آیتم‌های منو با دسته‌بندی مشخص
- 📝 **ثبت سفارش**: ثبت سریع سفارش‌ها و ارسال به آشپزخانه
- 🪑 **مدیریت میزها**: مشاهده وضعیت میزها (خالی، پر، رزرو شده)
- 📊 **داشبورد مدیریت**: مشاهده آمار فروش، سفارش‌های روز و ...
- 🧾 **صورتحساب**: محاسبه خودکار صورتحساب و چاپ رسید
- 👥 **مدیریت کاربران**: سطوح دسترسی مختلف برای مدیران، پیشخدمت‌ها و آشپزها
- 📈 **گزارش‌گیری**: گزارش فروش روزانه، هفتگی و ماهانه

## 🚀 نحوه نصب و اجرا

### پیش‌نیازها
- Node.js نسخه 18 یا بالاتر
- npm یا pnpm
- PostgreSQL

### مراحل نصب

```bash
# 1. کلون کردن پروژه
git clone https://github.com/hosseinkeyvani/cafe_manger.git

# 2. وارد شدن به پوشه پروژه
cd cafe_manger

# 3. نصب وابستگی‌ها
npm install

# 4. تنظیم متغیرهای محیطی
# فایل .env.example را کپی کرده و به .env.local تغییر نام دهید
cp .env.example .env.local

# 5. اجرای مهاجرت‌های دیتابیس
npx prisma migrate dev

# 6. اجرا در محیط توسعه
npm run dev
```

## 🎯 دستورات کاربردی

| دستور | توضیحات |
|-------|---------|
| `npm run dev` | اجرا در محیط توسعه |
| `npm run build` | ساخت نسخه نهایی |
| `npm start` | اجرای نسخه ساخته شده |
